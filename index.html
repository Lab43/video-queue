About

I built this website for teachers, but it should be useful to anyone wanting a simple way to share a collection of videos. Many teachers show videos in their classrooms, but websites like YouTube add links to “related videos” and other clutter that can be distracting and often isnâ€™t appropriate for a classroom. This website strips away as many of these distractions as possible, providing a cleaner and safer way to show videos.

Why do I still see ads?

Unfortunately, it's impossible prevent YouTube from showing ads in videos. I don't make any money from these ads, and I have no plans to put additional advertising on this website.

Open Source

This project is open source and hosted on GitHub. Bug fixes and feature contributions are always welcome.

This website uses HTML5 Boilerplate, FitText.js, FitVids.js, and Compass. It was designed in Sketch, coded in Sublime Text 2, and compiled with CodeKit. Typefaces are Oleo Script and Open Sans, served by Google Web Fonts. Header pattern is Debut Light by Luke McDonald.
